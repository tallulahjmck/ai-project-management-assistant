# AI Project Management Assistant

## Overview

The AI Project Management Assistant is a Retrieval-Augmented Generation (RAG) application designed to help project managers analyse project documentation, identify risks, generate recommendations and support decision-making.

The project demonstrates the practical application of Large Language Models (LLMs), vector databases and prompt engineering within a project delivery environment.

---

## Objectives

- Analyse project documentation using AI
- Identify project risks and dependencies
- Generate actionable recommendations
- Summarise complex project information
- Support project managers with decision-making
